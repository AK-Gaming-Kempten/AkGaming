Theme Reference

Light and Dark Theme Surface Matrix

This page shows the valid text and background pairings defined by the core theme, the highlight accent colors, and the supported nested surface stacks. Invalid combinations are intentionally omitted so this can be used as a reference while implementing UI.

Light Theme

Primary and secondary surfaces are light. Tertiary and quartiary surfaces are inverse.

Highlight Colors

Highlight tokens are accent colors for hover, focus, emphasis, and active states rather than general body text.

Highlight Primary

Use for focus rings, hover emphasis, selected accents, and lighter branded emphasis.

--color-highlight-primary Focus Ring Hover outline

Highlight Secondary

Use for primary actions, strong emphasis, active states, and confirmed-positive accents.

--color-highlight-secondary Primary CTA Accent rail

Dark Theme

Primary and secondary surfaces invert, while tertiary and quartiary swap into light inverse surfaces.
