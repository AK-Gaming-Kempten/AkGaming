## Regeln

### Game Jam Regeln

- Alle Spiele müssen im Zeitraum des Game Jams erstellt werden. Spiele, die vor dem 24. April
  2026 entwickelt wurden, werden nicht akzeptiert.
- Das Thema des Game Jams muss im Spiel eingebaut und erkennbar sein. Falls das nicht sofort klar
  ist, erklärt bitte bei der Abgabe kurz, wie ihr das Thema integriert habt.
- Spiele müssen über itch.io abgegeben werden, um zugelassen zu werden. Nur Spiele, die während
  des Jam-Zeitraums eingereicht wurden, können bewertet werden.
- Wenn es einen triftigen Grund für eine verspätete Abgabe gibt, könnt ihr uns auf Discord
  anschreiben und um Zulassung bitten.

### Spiel Regeln

- Spiele dürfen keine anstößigen, diskriminierenden oder nicht jugendfreien Inhalte enthalten.
  Kein USK 18.
- Abgegebene Spiele müssen nicht komplett fertig sein, sollten aber spielbar sein.
- Die Wahl der Game Engine ist frei.

### Asset Regeln

- Bevorzugt werden Assets, die während des Game Jams selbst erstellt wurden.
- Wenn fremde oder vor dem Game Jam erstellte Assets verwendet werden, müssen sie in einer Liste
  dokumentiert werden und ihr solltet die nötigen Nutzungsrechte besitzen.
- Dokumentation für fremde Assets: `Name [Zugriffsdatum]: Link (Was wurde verwendet? Wo wurde das Asset verwendet?)`
- Dokumentation für eigene, schon vorher erstellte Assets:
  `Name [Zugriffsdatum]: EIGENES (Was wurde verwendet? Wo wurde das Asset verwendet?)`
- Das Spiel sollte nicht zu mehr als 25 % aus fremden Assets bestehen.
- KI-generierte Assets oder KI-generierter Code dürfen nicht verwendet werden. Ausnahme:
  3D-Charakteranimation.
